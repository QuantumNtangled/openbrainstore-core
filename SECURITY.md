# Security Policy

## Reporting a vulnerability

Please **do not open a public issue** for security vulnerabilities.

Report privately via the contact form at
[openbrainstore.com/#contact](https://openbrainstore.com/#contact) (choose
"Something else" and mention security), or use GitHub's **private vulnerability
reporting** on this repository if enabled.

Include what you found, how to reproduce it, and the impact. We'll acknowledge
and work with you on a fix and disclosure timeline.

## Scope

This repository is the OpenBrainStore server. The hosted deployment at
openbrainstore.com is operated separately; reports about the hosted service are
also welcome through the same channel.
